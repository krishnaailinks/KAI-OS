KAI-OS codebase. They are kept for reference only.

| Folder           | Contents                                                        |
|------------------|-----------------------------------------------------------------|
| `legacy-sql/`    | Early-phase Supabase schema SQL scripts (pre-migration system). |
|                  | Superseded by `database/schema.sql` + `database/migrations/`.  |

These files are **not executed** in any part of the current deployment.
